## 기술 스택

| 분류 | 기술 |
|---|---|
| 프레임워크 | React 19 |
| 빌드 도구 | Vite 7 |
| 스타일링 | styled-components 6, Tailwind CSS 4 |
| 라우팅 | React Router DOM 7 |
| 아이콘 | react-icons 5 |

---

## 실행 방법

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```
